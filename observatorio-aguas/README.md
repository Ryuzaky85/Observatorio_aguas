# Observatorio de Aguas

Una aplicación web interactiva que permite explorar ríos, lagos y océanos en mapas satelitales y mostrar información ambiental y científica como niveles de contaminación, biodiversidad y usos humanos.

## Características

- Visualización de cuerpos de agua en mapas interactivos
- Información detallada sobre contaminación y biodiversidad
- Interfaz de usuario intuitiva y responsiva
- Datos organizados en tablas para fácil consulta

## Tecnologías Utilizadas

- React + Vite para el frontend
- Leaflet para mapas interactivos
- Tailwind CSS para estilos

## Instalación

```bash
# Clonar el repositorio
git clone [url-del-repositorio]

# Navegar al directorio del proyecto
cd observatorio-aguas

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## Uso

La aplicación tiene tres secciones principales:

1. **Mapa**: Visualiza los cuerpos de agua en un mapa interactivo
2. **Datos**: Muestra información detallada en formato tabular
3. **Acerca de**: Información sobre el proyecto y su propósito
